# Project Name: Statistical Analysis of Student Records

## Project Info
Date: September 2025
- Created by:
- Lavrenty "Larry" Orlov

## Overview
This program uses functions to read numerical records for five students from a .txt file, perform calculations based on that input data, and return the results of those calculations to a second .txt file.

## Features
- Makes computations and provides unique results based on data extracted from an external file
- Modular design with separate '.c' and '.h' files
- Easy to run in Visual Studio

## Tech Stack
- Language: C
- IDE: Visual Studio Community

## How to Run
1. Clone the repository on your device's command-line interface:
- ```git clone https://github.com/lavrentyorlov/StudentRecordsAnalysisPA3.git```
2. Open the project in Visual Studio.
3. Build the solution.
4. Run the program.
